Version 2.0 - The "ReBrand Update" (02/03/2026)
------------------------------------------
✨ NEW FEATURES:
• Streaming Mode allow to automatically throttles network to 1.0 Gbps on activation and restores your previous speed when disabled.
• Displays current monitor resolution and refresh rate of the Host PC
• Shows installed GPU model for quick hardware reference

🔧 IMPROVEMENTS:
• NetworkSpeedToggle has officially "evolved" into StreamTweak 💻<->💻
• Introduced a new app icon that better captures the new spirit: a fresh look for a smarter tool 🚀

Version 1.2.2 - The "Lightweight" Update (02/03/2026)
------------------------------------------
🔧 IMPROVEMENTS:
• Framework-dependent build: installer reduced from ~50MB to ~3MB
• Auto-installs .NET 8 runtime silently if not present on target machine

Version 1.2.1 - The "No Prerequisites" Update (01/03/2026)
------------------------------------------
🔧 IMPROVEMENTS:
• Switched to .NET 8 LTS for improved stability and compatibility
• Self-contained build: .NET 8 runtime bundled, no external prerequisites

Version 1.2 - The "UI Update" (01/03/2026)
------------------------------------------
✨ NEW FEATURES:
• Real-time current speed indicator
• Modern Windows 11 UI with rounded corners
• Complete Dark/Light theme (title bar + dropdowns) 
• Automatic Windows theme color detection

🔧 IMPROVEMENTS:
• Physical adapters only (no filters, VPNs)
• Persistent config (remembers adapter + theme)

------------------------------------------
Version 1.1 - The "Smart Update" (28/02/2026)
------------------------------------------
✨ NEW FEATURES:
• Full Graphic Settings UI (double-click or tray menu)
• Automatic physical Ethernet adapter detection
• Automatic driver limitation bypass for Link Speeds
• Native Windows Toast Notifications for confirmation
• Intelligent polling waits for router handshake
• Real-time link speed reading via kernel APIs

🔧 IMPROVEMENTS:
• Replaced double-click toggle with Settings panel
• Filters out virtual adapters, Wi-Fi, Bluetooth
