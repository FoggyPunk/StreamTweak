Version 3.1.0 - The "Atmos Update" (06/03/2026)
------------------------------------------
✨ NEW FEATURES:
• Audio tab — Auto Dolby Atmos for Headphones: when a streaming session
  is detected and stays active for at least 30 seconds, StreamTweak
  automatically enables Dolby Atmos for Headphones on Steam Streaming
  Speakers, provided Dolby Access is installed and the format is available
• Audio tab — Dolby Access detection indicator: a live status indicator
  shows whether Dolby Atmos for Headphones is detected on the host PC,
  checked via the Windows Spatial Audio API at startup and each time the
  Audio tab is opened
• Audio tab — activation status: the Status box turns green when Dolby
  Atmos for Headphones is successfully enabled
• Tray right-click — Dolby Atmos toggle: enable or disable the Dolby
  Atmos auto-activation directly from the system tray context menu,
  in sync with the Audio tab toggle

🔧 IMPROVEMENTS:
• About tab — version check: the current version of StreamTweak is now
  compared against the latest GitHub release; if a newer version is
  available, a download link is shown automatically

Version 3.0.0 - The "Awareness Update" (06/03/2026)
------------------------------------------
✨ NEW FEATURES:
• Logs tab: browse the last 10 speed-change sessions at a glance,
  including trigger mode (Auto/Manual), duration, and original speed
• Streaming app detection in Logs tab: StreamTweak shows whether it has
  correctly located the log folder of the streaming server installed on
  the host PC (Sunshine, Apollo, Vibeshine or Vibepollo); click the path
  to open the folder in Explorer
• About tab: version info, GitHub repository link, GPL v3 license badge,
  and PayPal donation button all in one place
• Tray right-click — Auto Mode toggle: enable or disable Auto Streaming
  Mode directly from the system tray without opening Settings
• Tray speed readout: current link speed of the selected adapter is
  always visible in the tray context menu
• Tray streaming status: the context menu now shows whether a streaming
  session is currently active or inactive

🔧 IMPROVEMENTS:
• Full UI revision: centered layout across all panels, redesigned tab
  bar with Dark/Light Mode button integrated, improved spacing and
  visual balance throughout the Settings window

Version 2.5.2 - The "Vibe Update" (06/03/2026)
------------------------------------------
✨ NEW FEATURES:
• Vibepollo and Vibeshine support: Auto Streaming Mode now works with
  all major Sunshine forks
• Dynamic log discovery: streaming server location is auto-detected via
  registry lookup, with Program Files scan as fallback — custom
  installation paths fully supported
• Log rotation support: automatically follows dynamic log files
  (sunshine-YYYYMMDD-HHMMSS-mmm.log) used by Vibepollo and Vibeshine

🔧 IMPROVEMENTS:
• StreamingLogMonitor re-runs full log discovery every 10 seconds,
  catching dynamic log files that appear after startup
• Updated UI warning: disconnect during speed adjustment is now stated
  clearly, with 30-second reconnect window explained

🐛 STABILITY & FIXES:
• Fixed monitor not switching to dynamic log files when they appear
  after StreamTweak has already started
• Fixed FileStream disposed too early in StreamingLogMonitor

Version 2.5.1 - The "Silent Power Update" (04/03/2026)
------------------------------------------
✨ NEW FEATURES:
• StreamTweakService: Windows Service (LocalSystem) handling all network speed
  changes with elevated privileges -- no UAC prompts, ever
• Named Pipe IPC: secure communication between app and service, keeping the
  privilege boundary clean and UX uninterrupted

🔧 IMPROVEMENTS:
• No UAC prompts in Manual Mode, Auto Mode, and tray menu
• Automatic fallback to UAC prompt if the service is unavailable
• Service lifecycle fully managed by the installer

🐛 STABILITY & FIXES:
• Minor bug fixes
